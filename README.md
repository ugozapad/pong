# Pong

A game written in ancient C++98 with Direct3D 9.

![Preview](/data/textures/sshot.png)

## How to build (Windows-only)

1. **Clone the repository**
	```bash
	git clone https://github.com/ugozapad/pong
	cd pong
	```
2. **Generate projects from CMake**
	```bash
	cmake -B build
	```
3. **Running**
	```
	Set working directory to the repo root. (i.e. ../) and from Visual Studio
	```
